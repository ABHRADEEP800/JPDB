How to Use This System
  1. Enter a student's Roll Number to search for existing records
  2. If the student exists, their information will load and you can Update it
  3. If the student doesn't exist, all fields will unlock for new entry
  4. Click Save to store new records or Update to modify existing ones
  5. Use Reset to clear the form and start over
